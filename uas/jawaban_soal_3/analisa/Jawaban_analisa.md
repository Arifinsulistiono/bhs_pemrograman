# JAWABAN

## 3
    GUI (Graphical User Interface) adalah antarmuka pengguna yang memungkinkan interaksi dengan perangkat lunak melalui elemen visual seperti jendela, tombol, dan menu. GUI membuat aplikasi lebih mudah digunakan karena pengguna dapat berinteraksi dengan aplikasi secara langsung melalui elemen grafis.

## contoh GUI menggunakan tkinter
    Tkinter adalah pustaka standar Python yang digunakan untuk membuat antarmuka grafis. Di bawah ini adalah contoh kode sederhana menggunakan Tkinter untuk membuat jendela dengan tombol yang, ketika ditekan, menampilkan pesan.
